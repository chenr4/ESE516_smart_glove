# Smart Training Glove Website

This folder contains a polished public GitHub Pages website for the ESE 5160 final project.

## Files

- `index.html` — main website page
- `assets/` — project images used by the website

## How to deploy

1. Upload `index.html` and the `assets/` folder to your final submission GitHub repository.
2. In GitHub, go to `Settings → Pages`.
3. Select `Deploy from a branch`.
4. Choose `main` and `/root`.
5. Save and wait for GitHub Pages to publish the site.
